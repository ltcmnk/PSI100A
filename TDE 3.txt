#1. Imprima os números de 1 até 99, com incremento de 2. Exemplo: 1, 3, 5.....97, 99

num = 1
while num <= 99:
    print(num)
    num = num + 2





#2. Imprima os números de 50 até 0 com decremento de 5. Exemplo: 50, 45, 40.....5, 0

num = 50
while num >= 0:
    print(num)
    num = num - 5





#3. Imprima os números de -100 até 100, com incremento de 10. Exemplo: -100, -90, -80.....90, 100

num = -100
while num <= 100:
    print(num)
    num = num + 10





#4. Imprima os números múltiplos de 4 existentes no intervalo aberto de 1 a 100.

num = 4
while num < 100:
    print(num)
    num = num + 4





#5. Imprima os números ímpares de 1 até n, sendo n fornecido pelo usuário.

num = int(input("Digite um número inteiro para imprimir os números ímpares entre [1] e ele: "))
numero = 1

while numero <= num:
    print(numero)
    numero = numero + 2




#6. Imprima uma tabela de conversão de polegadas para centímetros, cuja escala vai de
#1 a 20 polegadas. A conversão entre estas duas unidades é dada por: polegada = centímetro × 2,54


polegada = float(input("Digite o valor para conversão de polegadas para centímetros: "))
if 1 <= polegada <= 20:
    centimetro = polegada / 2.54
    print("A conversão fica igual a", centimetro)
else:
    print("Valor inválido! A escala vai de 1 a 20 polegadas.")
